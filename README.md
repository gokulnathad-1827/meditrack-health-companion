# 🩺 MediTrack

A full-stack HealthTech application for tracking daily vitals, managing medications, recording symptoms, scheduling doctor visits, and generating health reports.

## Features

- User Registration & Login
- User Profile Management
- Daily Vitals Tracking
- Medicine Reminder
- Symptom Diary
- Doctor Visit Records
- Prescription Upload
- Health Analytics Dashboard
- PDF Health Report Generation
- Emergency SOS

## Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- React Router
- Axios
- Recharts

### Backend
- Spring Boot
- Spring Data JPA
- REST API

### Database
- MySQL
